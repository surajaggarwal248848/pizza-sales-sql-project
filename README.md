# Pizza Sales Analysis Using SQL

## Overview
Analyzed pizza sales data using SQL to identify sales trends and business insights.

## Tools Used
- SQL
- Microsoft Excel

## Key Analysis
- Total Revenue
- Total Orders
- Average Order Value
- Sales by Category and Size
- Top and Bottom Selling Pizzas
- Hourly and Daily Sales Trends

## Outcome
Generated actionable insights to support business decision-making using data analysis techniques.
